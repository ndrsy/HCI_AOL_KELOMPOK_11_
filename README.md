# AOL HCI UNIVEXUS

## Deskripsi
Website ini dibuat menggunakan *HTML, CSS, dan JavaScript* sebagai tugas akhir AOL. 
Halaman yang tersedia:
- index.html → halaman utama
- login.html → halaman login
- register.html → halaman registrasi
- forum.html → forum diskusi
- career-match.html → pencocokan karir
- scholarship-calculator.html → kalkulator beasiswa
- results.html → hasil perhitungan
- specifications.html → spesifikasi sistem

## Struktur Folder
- css/ → styling
- js/ → interaktivitas (auth.js, forum.js, login.js, dll.)
- images/ → aset gambar kampus (BINUS, ITB, UI, UGM, dll.)

## Cara Menjalankan
1. Clone repository ini atau download ZIP.
2. Buka file index.html di browser.
3. Navigasi ke halaman lain melalui menu.
